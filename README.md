# Alien Cat - Website

Sitio web oficial del bot de Discord **Alien Cat**.

## Descripcion

Pagina web profesional para el bot de moderacion Alien Cat. Incluye:

- Seccion Hero con avatar del bot
- Caracteristicas del bot
- Lista de comandos organizados por categorias
- Roadmap de proximas actualizaciones
- Estadisticas animadas
- Links de invitacion y servidor de soporte

## Estructura del Proyecto

```
website/
├── index.html              # Pagina principal
├── server.py               # Servidor Python (desarrollo)
├── assets/
│   ├── css/
│   │   └── style.css       # Estilos CSS
│   ├── js/
│   │   └── main.js         # JavaScript
│   └── images/
│       └── bot-avatar.jpg  # Avatar del bot
└── README.md
```

## Hosting

Este sitio web es estatico y puede ser hosteado en:

- **GitHub Pages** - Gratis, recomendado
- **Netlify** - Gratis
- **Vercel** - Gratis

### Para GitHub Pages:

1. Sube los archivos de la carpeta `website/` a un repositorio
2. Ve a Settings > Pages
3. Selecciona la rama `main` y la carpeta raiz
4. Tu sitio estara disponible en `https://tu-usuario.github.io/nombre-repo/`
